Compiler Design Visualizer

Project Description

This project is an interactive web-based tool that visualizes different phases of a compiler such as lexical analysis, syntax analysis, semantic analysis, intermediate code generation, optimization, and target code generation.

 Features

- Lexical Analysis (Token generation)
- Syntax Analysis (Grammar validation)
- Semantic Analysis (Meaning checking)
- Intermediate Code Generation
- Code Optimization
- Target Code Generation

 Technologies Used

- HTML
- CSS
- JavaScript

 Objective

To help understand compiler design concepts through visualization.
